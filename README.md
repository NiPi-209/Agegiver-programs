# About
These two programs were made as a sort of challenge, where I wanted to see how small I could make a python program that gives your age based on inputs.

The first one (Agegiver) is that I did the challenge in.
The second one (Agegiver AE) is a version where I added more features to make it better, without it being overwelming.

The only requirement are:

Atleast 249 Bytes (Agegiver) or 4,15 KB (Agegiver AE) (if you are conserned about this, are you using an Apple ][e or something?)

Python 3.11 or better

That's it :)
